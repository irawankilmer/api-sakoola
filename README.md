## api-sakoola

## install
- Clone project
```gitexclude
git clone https://github.com/irawankilmer/api-sakoola.git
```

- Install
```gitexclude
cd api-sakoola
```
```gitexclude
npm install
```
- Setting<br>
Buat file .env lalu isinya copy semua dari file .env.example<br>
Sesuaikan value-nya
- Generate prisma
```gitexclude
npx prisma generate
```
- run project
```gitexclude
npm run dev
```
